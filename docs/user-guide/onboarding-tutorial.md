# Onboarding Tutorial

## Landlord Onboarding

### Step 1: Create Your Account (5 minutes)

**What You'll Do:**
Create your landlord account and set up your profile.

**Instructions:**
1. Go to the platform homepage
2. Click **"Sign Up"**
3. Select **"Landlord"** as your account type
4. Fill in your information:
   - Full Name: Your legal name
   - Email: Your primary email address
   - Phone: Your contact number
   - Password: Create a strong password
   - Company Name: Your business name (optional)
5. Click **"Create Account"**
6. Check your email and click the verification link
7. Log in to your new account

**✅ Success Check:**
- You can log in to your account
- Your dashboard is visible
- You see the landlord navigation menu

---

### Step 2: Complete Your Profile (5 minutes)

**What You'll Do:**
Add important details to your profile.

**Instructions:**
1. Click your profile picture in the top right
2. Select **"Profile Settings"**
3. Upload a profile picture (optional but recommended)
4. Add your company information:
   - Company name
   - Business address
   - Tax ID (for reporting)
5. Add emergency contact information
6. Click **"Save Changes"**

**💡 Pro Tip:**
A complete profile builds trust with tenants and makes you look professional.

---

### Step 3: Set Up Payment Processing (10 minutes)

**What You'll Do:**
Connect your payment processor to accept rent payments.

**Instructions:**
1. Go to **Settings** > **Payment Settings**
2. Click **"Connect Stripe Account"**
3. Follow Stripe's setup process:
   - Provide business information
   - Add bank account for deposits
   - Verify your identity
4. Configure payment options:
   - Enable credit/debit cards
   - Enable ACH bank transfers
   - Set convenience fees (optional)
5. Set up late fee rules:
   - Grace period (e.g., 5 days)
   - Late fee amount or percentage
   - Maximum late fee
6. Click **"Save Payment Settings"**

**✅ Success Check:**
- Stripe account shows as "Connected"
- Payment methods are enabled
- Late fee rules are configured

---

### Step 4: Add Your First Property (15 minutes)

**What You'll Do:**
Add a property to your portfolio.

**Instructions:**
1. Navigate to **Properties**
2. Click **"Add Property"**
3. Fill in basic information:
   - Property address
   - Property type (house, apartment, condo, etc.)
   - Year built
4. Add property details:
   - Bedrooms: Number of bedrooms
   - Bathrooms: Number of bathrooms
   - Square footage
   - Parking spaces
5. Select amenities (check all that apply):
   - Pool, gym, laundry, dishwasher, etc.
6. Set rent information:
   - Monthly rent amount
   - Security deposit
   - Pet deposit (if applicable)
7. Write a property description:
   - Highlight key features
   - Mention nearby attractions
   - Include any special terms
8. Upload photos:
   - Drag and drop or click to upload
   - Add at least 5-10 photos
   - First photo will be the cover image
9. Set property status to **"Available"**
10. Click **"Save Property"**

**💡 Pro Tips:**
- Use high-quality photos taken in good lighting
- Include photos of all rooms, exterior, and amenities
- Write a detailed, honest description
- Highlight what makes your property special

**✅ Success Check:**
- Property appears in your property list
- Property status shows as "Available"
- Photos are displayed correctly
- Property appears in tenant search

### Step 5: Invite Your First Tenant (10 minutes)

**What You'll Do:**
Invite an existing tenant or prepare for new tenant applications.

**Option A: Invite Existing Tenant**

1. Go to **Tenants**
2. Click **"Invite Tenant"**
3. Enter tenant's email address
4. Select the property
5. Add a personal message (optional):
   ```
   Hi [Tenant Name],
   
   Welcome to our new property management system! This platform will make it easy for you to pay rent, submit maintenance requests, and communicate with me.
   
   Please create your account using this invitation.
   
   Best regards,
   [Your Name]
   ```
6. Click **"Send Invitation"**
7. Tenant receives email to create account

**Option B: Prepare for New Applications**

1. Ensure property is set to "Available" status
2. Property will appear in tenant search
3. You'll receive notifications when applications are submitted

**✅ Success Check:**
- Invitation sent successfully
- Tenant receives email
- You can see pending invitation in tenant list

### Step 6: Create Your First Lease (20 minutes)

**What You'll Do:**
Create a lease agreement using the Lease Creation Wizard.

**Prerequisites:**
- Property added
- Tenant invited and account created

**Instructions:**

**Step 1: Property Selection**
1. Go to **Leases** > **Create Lease**
2. Select your property
3. Select unit (if applicable)
4. Click **"Next"**

**Step 2: Tenant Selection**
1. Select your tenant from the list
2. Add co-tenants if applicable
3. Verify contact information
4. Click **"Next"**

**Step 3: Lease Terms**
1. Set lease dates:
   - Start date: When tenant moves in
   - End date: When lease expires (typically 12 months later)
2. Enter monthly rent amount
3. Enter security deposit amount
4. Enter pet deposit (if applicable)
5. Set late fee policy:
   - Grace period (e.g., 5 days)
   - Late fee amount
6. Choose lease type:
   - Fixed-term (specific end date)
   - Month-to-month (ongoing)
7. Click **"Next"**

**Step 4: Additional Terms**
1. Select utilities included (if any):
   - Water, sewer, trash
   - Electric, gas
   - Internet, cable
2. Specify parking:
   - Number of spaces
   - Parking location
3. Set pet policy:
   - Pets allowed: Yes/No
   - Types allowed
   - Number of pets
   - Pet rent/deposit
4. Set other policies:
   - Smoking policy
   - Subletting policy
   - Guest policy
5. Add custom clauses (optional)
6. Click **"Next"**

**Step 5: Review and Sign**
1. Review all lease terms carefully
2. Preview the lease document
3. Make any necessary edits
4. Click **"Send for Signature"**
5. Set signing order:
   - Sequential: You sign first, then tenant
   - Parallel: Both can sign at any time
6. Add a message to tenant
7. Click **"Send"**

**What Happens Next:**
- You and tenant receive email notifications
- You can sign the lease electronically
- Tenant signs after you (if sequential)
- Once both parties sign, lease becomes active
- Signed lease is automatically saved

**💡 Pro Tip:**
Review the lease carefully before sending. Once signed, changes require an addendum.

**✅ Success Check:**
- Lease created successfully
- Signature requests sent
- You can track signature status
- Lease appears in your lease list

### Step 8: Customize Notification Preferences (5 minutes)

**What You'll Do:**
Set up how you want to receive notifications.

**Instructions:**

   - **Payment notifications**: Email + In-app
   - **Maintenance requests**: Email + In-app + SMS

   - **Messages**: In-app (Email for important)
   - **Lease events**: Email + In-app
4. Set quiet hours: 10:00 PM - 7:00 AM
5. Email digest: Daily summary
6. Click **"Save Preferences"**

**✅ Success Check:**
- Notification preferences saved
- You receive test notification
- Quiet hours configured

### Step 4: Submit an Application (20 minutes)

**What You'll Do:**
Apply for a property you're interested in.

**Before You Start:**
Gather these documents:
- Photo ID (driver's license or passport)
- Recent pay stubs (last 2-3 months)
- Employment verification letter
- Bank statements (last 2-3 months)
- Previous landlord contact information
- Personal references
- Pet vaccination records (if you have pets)

**Instructions:**

1. Find a property you want to apply for
2. Click **"Apply Now"**
3. Complete the application form:

**Personal Information:**
- Full legal name
- Date of birth
- Social Security Number
- Contact information
- Current address

**Employment:**
- Employer name and address
- Job title
- Start date
- Monthly income
- Supervisor contact

**Rental History:**
- Previous landlord information
- Previous address
- Dates of tenancy
- Reason for leaving

**References:**
- 2-3 personal references
- Names, relationships, contact info

**Additional Information:**
- Number of occupants
- Pets (if any)
- Vehicles
- Emergency contact

4. Upload required documents:
   - Click "Upload" for each document type
   - Select files from your computer
   - Ensure files are clear and readable

5. Review background check authorization:
   - Read carefully
   - Check boxes to consent
   - Electronic signature

6. Pay application fee:
   - Enter payment information
   - Review fee amount
   - Submit payment

7. Review entire application:
   - Check all information for accuracy
   - Edit any sections if needed
   - Click **"Submit Application"**

8. Receive confirmation:
   - Confirmation email sent
   - Application tracking number provided

**💡 Pro Tips:**
- Be thorough and honest
- Double-check all information
- Upload clear, legible documents
- Respond quickly if additional info is requested

**✅ Success Check:**
- Application submitted successfully
- Confirmation email received
- Application appears in your dashboard
- Status shows as "Submitted"

## Troubleshooting Common Issues

### Can't Log In
1. Check email and password
2. Use "Forgot Password" if needed
3. Clear browser cache
4. Try different browser
5. Contact support

### Payment Failed
1. Check payment method
2. Verify sufficient funds
3. Update card expiration date
4. Try different payment method
5. Contact support

### Can't Upload Files
1. Check file size (under 10MB)
2. Check file format
3. Try different browser
4. Check internet connection
5. Contact support

### Not Receiving Notifications
1. Check spam folder
2. Verify email address
3. Check notification settings
4. Add to email contacts
5. Contact support
