# ICO State Components

Components representing different ICO lifecycle states and dashboard tabs.

| File | Responsibility |
|------|----------------|
| ActiveICO.tsx | Main purchase interface during active ICO phase |
| ActivePresale.tsx | Purchase interface during presale phase |
| DashboardICOCountdown.tsx | Dashboard view with countdown before ICO starts |
| OverviewTab.tsx | Dashboard tab showing portfolio and earnings summary |
| PostICODashboard.tsx | Main dashboard container after ICO ends |
| PresaleCountdown.tsx | Countdown display before presale begins |
| RewardsTab.tsx | Dashboard tab displaying staking and fee rewards |
| TokenomicsTab.tsx | Dashboard tab showing token distribution charts |
| WhitepaperTab.tsx | Dashboard tab with whitepaper content and download |
