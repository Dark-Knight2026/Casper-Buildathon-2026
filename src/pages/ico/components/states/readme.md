# ICO State Components

Components representing different ICO lifecycle states and dashboard tabs.

| File | Responsibility |
|------|----------------|
| PrivateSaleActive.tsx | Active private sale page with countdown and purchase form |
| PrivateSaleCountdown.tsx | Countdown display before private sale starts |
| PostICODashboard.tsx | Container for post-ICO tabs navigation |
| OverviewTab.tsx | Dashboard showing balance, staking, and earnings |
| RewardsTab.tsx | Details staking rewards and transaction fees |
| TokenomicsTab.tsx | Displays token supply, vesting, and allocation chart |
| WhitepaperTab.tsx | PDF viewer with sidebar for whitepaper sections |
