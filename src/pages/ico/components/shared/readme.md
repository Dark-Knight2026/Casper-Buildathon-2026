# ICO Shared Components

Reusable UI components shared across ICO states and pages.

| File | Responsibility |
|------|----------------|
| AmountInput.tsx | Handles token purchase amount input with validation |
| Card.tsx | Provides styled card container with ICO theme |
| CountdownTimer.tsx | Displays countdown to ICO events with time units |
| CurrencySelector.tsx | Dropdown selector for payment currency selection |
| DashboardTabs.tsx | Tab navigation component with animated underline |
| InfoCard.tsx | Grid container for displaying key statistics |
| MainButton.tsx | Primary action button with ICO styling |
| ProgressBar.tsx | Visual progress indicator for token sales |
| SubTitle.tsx | Styled subtitle text component |
| Title.tsx | Styled heading text component |
| TransactionHistory.tsx | Lists user transaction history with status |
| UserTokenBalance.tsx | Displays user's token balance summary |
| VestingProgressBlock.tsx | Shows vesting schedule and unlock progress |
| WalletCard.tsx | Wallet connection and token purchase interface |
