# Component Tests

Unit tests for reusable feature components in `src/components/`.

| File | Responsibility |
|------|----------------|
| FeaturedProperties.test.tsx | Tests featured property cards — rendering from `FEATURED_PROPERTIES`, click / Enter / Space navigation, favorite-button stopPropagation, keyboard focusability |
| LandingHeader.test.tsx | Tests the public landing-page header — brand link, Properties / Token Sale nav links, Sign In / Get Started auth CTAs |

Subdirectories `help/`, `ico/`, `profile/`, `tenant/`, and `ui/` document their own test files in their respective `readme.md` files.
